#Vorgabe:
Aufgabe ist es es Daten von Radfahrern mit Hilfe eines Sortier-Algorithmus zu sortieren und als Power Curve Grafik aus zu geben.

#Installationsanleitung:
Bevor das Programm verwendet werden kann, muss das Repository auf Ihr Gerät geklont ( Befehl: git clone https://github.com/luggi003/Leistungskurve-1.git) und anschließend mit PDM installiert werden ( Befehl: pdm install ).
